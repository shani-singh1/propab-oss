Before accepting any claim — your own hypothesis, a prior finding, or a fresh result —
put it through a structured critique. Adapted from formal scientific critical-thinking
practice; the point is to catch the ways a result can look convincing while being wrong.

1. **Separate data from interpretation.** State exactly what was observed vs. what it is
   claimed to mean. Most errors hide in the leap between the two. Ask: do the conclusions
   actually follow from the data, or is there an unstated inferential jump?

2. **Enumerate alternative explanations.** For any apparent effect, list the competing
   accounts and rule them out explicitly:
   - **Confounding** — is there a variable that drives both the predictor and the outcome?
     Was it measured and controlled? A predictor that merely proxies the group label is a
     confound, not a mechanism.
   - **Reverse causation** — could the outcome cause the "cause"? Check temporality: does
     the proposed cause precede the effect?
   - **Selection / leakage** — did the way cases were chosen, or information crossing the
     train/test boundary, manufacture the effect?

3. **Judge causal strength honestly.** Can the design support a causal claim, or only a
   correlational one? Do not upgrade a correlation to a mechanism without a manipulation
   or a hold-out that breaks the confound.

4. **Interrogate the statistics.** Sufficient sample/power? Correct test for the data?
   Multiple-comparisons handled? Effect size reported alongside significance (a tiny p on a
   trivial effect is not a finding)? Was the null a real adversarial control, not a
   just-so story?

5. **Name the fallacy if present.** Unfalsifiability (an untestable claim), circularity
   (using the answer to prove the answer), overgeneralization beyond the tested scope,
   cherry-picking. If the claim commits one, say so and downgrade it.

6. **Grade the evidence.** Rate the strength (strong / suggestive / weak / none) and let
   the wording match it. A result that survives this critique earns a confident claim; one
   that does not is reported as inconclusive or as a rediscovery — never oversold.

Run this critique on your OWN candidate before proposing it: if you can already see the
confound or the alternative explanation, redesign the test now.
